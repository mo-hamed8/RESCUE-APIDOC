API Documentation

Register

Method: POST

Endpoint: /api/register

Headers:

Content-Type: application/json

Request Body:

{
    "name":"mohammed",
    "email":"moqs1dd@gmail.com",
    "password":"123123123",
    "password_confirmation":"123123123"
}

Response Body:

{
    "id": 7,
    "name": "mohammed",
    "email": "moqs1dd@gmail.com"
}
